## What's in this repo

This is the source code for the **NeolithAI Agency website** — a fully static, multi-page site with no frameworks or build tools required.

### Tech stack
- Pure HTML5 + CSS3 + Vanilla JS
- Shared CSS/JS across all pages (`neolithai-shared.css`, `neolithai-shared.js`)
- Netlify Forms for contact & newsletter
- Canvas API for animated hero visualization
- AI chat widget powered by Claude API (Nео assistant)
- Deployed on Netlify with auto-deploy from this repo

| File | Description |
